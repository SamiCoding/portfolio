장고캣

깃허브 : 깃허브로 이동
1차 진행(개별) : 2022.09 ~ 2022.11 2차 진행(개별) : 2023.10 ~ 2023.11

  “장고캣”은 게시판 기능을 가진 간단한 웹 사이트 입니다.

  코딩 학원에서 근무할 때 수업용으로 만들었던 웹 사이트 입니다. 학생들이 배운 Python을 활용할 수 있도록 웹 프레임워크를 Django로 선택하였습니다.

  CRUD를 기본으로 하는 글과 댓글 작성이 가능한 게시판을 메인으로 하였습니다. 그리고 학생들이 재밌게 즐길 수 있도록 간단한 웹 게임 컨텐츠를 추가하여 스스로 코딩해볼 수 있도록 만들었습니다.

Frontend : Django Templates
Backend : Django
Database : SQLite
Deployment : PythonAnywhere

그려줘 친구들!

웹 사이트 : 프로젝트로 이동
1차 진행(팀) : 2020.05 ~ 2020.06 2차 진행(개별) : 2023.07 ~ 2023.09

  “그려줘 친구들!”은 인공지능을 활용한 아동 교육용 게임입니다. 아이들이 직접 그림을 그리면서 동화에 참여하는 느낌을 줄 수 있도록 만들었습니다.

  SSAFY에서 위 프로젝트를 처음 진행할 당시 프론트엔드를 맡았습니다. 이야기를 등록하는 관리자 페이지, 게임 진행 그리고 게임을 통해 그려진 작품들을 볼 수 있는 마이페이지 기능을 구현하였습니다. Jira를 사용해서 팀원들과 프로젝트 진행 상항을 공유하였고, Git을 이용하여 협업했습니다.

  이후 웹 사이트에 새로운 기능을 추가하기 위해 개별적으로 프로젝트를 진행하였습니다. 작품 꾸미기 등의 기능을 추가하기 위해 이전에는 담당하지 않았던 백엔드도 함께 다루었습니다.

Frontend : Vue.js, Vuetify
Backend : Django
Database : PostgreSQL
Deployment : Vercel(Frontend), Railway(Backend)

슬기로운 음식처방

진행(팀) : 2020.03 ~ 2020.04

  “슬기로운 음식처방”은 사용자의 건강상태와 맞는 음식점을 동의보감 데이터를 활용해 추천해주는 웹 사이트 입니다.

  KNN 알고리즘을 이용해 사용자에게 필요한 재료에 해당하는 음식점을 추천해 주도록 구현하였습니다. 그리고 KoNLPy를 활용하여 음식점의 댓글을 분석 후 음식점에 대한 키워드를 추출하였습니다.

Frontend : Vue.js, Vuetify
Backend : Django
Database : SQLite
Deployment : AWS

Vote Us

진행(팀) : 2020.01 ~ 2020.02

  “Vote Us”는 사용자가 얼굴인식과 지문인식을 통한 인증을 완료한 후 투표를 할 수 있도록 구현한 웹 사이트 입니다.

  새로운 경험을 위해 지금까지 사용해보지 않았던 React를 사용해 보았습니다. 카메라와 지문 센서를 이용하여 사용자의 얼굴과 지문 정보 얻은 후 데이터를 Flask로 전달하도록 구현하였습니다. 그리고 본인인증을 완료한 이후 투표 기능이 활성화되도록 만들었습니다.

Frontend : React
